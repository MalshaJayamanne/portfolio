# Personal Developer Portfolio 🚀

A modern, high-performance personal developer portfolio built with a bleeding-edge tech stack to showcase projects, skills, and experience with a stunning UI.

## 🛠️ Tech Stack

This project leverages the best modern web development tools:

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Routing**: [TanStack Router](https://tanstack.com/router/latest) / [TanStack Start](https://tanstack.com/start/latest)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) (Headless accessibility)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Data Visualization**: [Recharts](https://recharts.org/)
- **State Management & Fetching**: [TanStack Query](https://tanstack.com/query/latest)

## ✨ Features

- **Blazing Fast Performance**: Powered by Vite and optimized React 19.
- **Accessible UI**: Radix UI primitives ensure full keyboard navigation and screen reader support.
- **Fluid Animations**: Meaningful micro-interactions and page transitions powered by Framer Motion.
- **Bento-style Layouts & Charts**: Clean visual data presentation of skills and metrics using Recharts.
- **Fully Responsive**: Mobile-first Tailwind utility approach guaranteeing great experience across all devices.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn or pnpm

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repo-url>
   cd portofolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173` (or the port Vite provides).

### Building for Production

To create an optimized production build:

```bash
npm run build
```

You can then preview the production build locally:

```bash
npm run start
```

## 📁 Project Structure

- `components/` - Reusable UI components (buttons, cards, layout pieces).
- `routes/` - Route definitions for TanStack router.
- `styles.css` - Global CSS styles and Tailwind imports.
- `lib/` - Utility functions and helpers.
- `hooks/` - Custom React hooks.
- `assets/` - Static files like images and icons.
